Unity Game Development Partner

2nyはUnityに特化した開発組織として、コンソールからモバイルまでの大規模タイトルで求められるレンダリングとゲームロジックの高度な要件に対応します。

HDRP・URPを軸とした表現開発、Addressablesを活用したアーキテクチャ設計、CI/CDと品質保証を含む開発プロセス整備まで、エンタープライズ水準のパートナーシップを提供します。

相談する

提供サービス

  • Unityアーキテクチャ設計と技術リード（HDRP・URP・SRP、Timeline、Shader Graph、パフォーマンス最適化）
  • 描画・ゲームプレイにおける技術検証と課題解決、ツール開発、マルチプラットフォーム展開（iOS/Android/PC/コンソール）
  • 開発基盤構築と運用支援（CI/CD、自動テスト、コード品質管理）

プロジェクト実績

NDAに基づき詳細非公開ですが、大規模なUnityタイトルにおいて開発組織の技術リードおよび品質保証を担当し、リリースまで一貫して支援した実績があります。

お問い合わせ

新規開発、既存タイトルの改善、技術アセスメントなど、Unityに関する課題のご相談を承っています。要件やタイムラインが未確定の場合もお気軽にお知らせください。

  • 通常1～2営業日以内に担当者よりご連絡いたします。
  • オンラインミーティング、Slackなどでの共同作業にも対応しています。
